# Harves Entertainment Website

Static responsive website for Harves Entertainment.

## Files

- `index.html` - main page
- `styles.css` - responsive layout and animation styles
- `script.js` - scroll, menu, and interaction effects
- `assets/` - image assets
- `.nojekyll` - keeps GitHub Pages from processing the site with Jekyll

## GitHub Pages

Upload every file in this folder to the root of a GitHub repository.

Then open:

Settings -> Pages -> Build and deployment -> Deploy from a branch

Choose:

- Branch: `main`
- Folder: `/root`

After GitHub finishes deployment, the site will be available from the Pages URL shown in the repository settings.
